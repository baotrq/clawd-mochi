Drop your animated background recording here.

Accepted filenames (the app tries them in this order):
  room.webm   ← preferred (smaller file, better quality)
  room.mp4    ← fallback (works everywhere)

Either one is enough; if you have both, webm is used first.

Tips:
- Record/export it as a clean loop if you can (no hard cut at the seam).
- Muted autoplay loop, so audio doesn't matter.
- Keep it reasonably sized; huge files hurt load time.

Until a video is present here, the page shows the still room.png frame.
